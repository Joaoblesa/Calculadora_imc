# 📱 Calculadora de IMC - Jetpack Compose

Uma calculadora de Índice de Massa Corporal (IMC) moderna e minimalista, desenvolvida com **Android Jetpack Compose**. O projeto foca em uma interface limpa, utilizando componentes de Material Design 3.

## 🚀 Funcionalidades

* Cálculo de IMC em tempo real.
* Interface responsiva com `Column`, `Row` e `Spacer`.
* Botão estilizado com `OutlinedButton` e bordas arredondadas.
* Design focado na experiência do usuário (UX).

## 🛠️ Tecnologias Utilizadas

* **Kotlin**: Linguagem principal.
* **Jetpack Compose**: Toolkit moderno para construção de UI nativa.
* **Material Design 3**: Padrões de design mais recentes do Google.
* **Compose Modifiers**: Para controle preciso de layout e espaçamento.

## 📸 Demonstração do Layout

O layout principal consiste em:
1.  Exibição do resultado (Valor do IMC e Classificação).
2.  Botão de ação personalizado ("CALCULAR").
